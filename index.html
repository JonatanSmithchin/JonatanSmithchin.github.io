网络编程

网络通信协议

  • TCP Transmission Control Protocol
    • 面向连接的通信协议,连接后要进行 “三次握手”确认连接 可靠 。
    • 三次握手后,Client与Server间可进行大量数据的传输。虽然效率较低，但可保证传输的安全。
  • UDP User Datagram Protocol
    • 消耗资源少,传输效率较高,传输文件有大小限制（64K）。
    • 不能保证数据传输的完整性。

Mysql学习

发表于 2022-03-22

Mysql子句

  • Mysql默认不区分大小写,输入时应保证输入格式的统一,命令子句可以全部大写;
  • 所有命令以”;”结尾;

HELP

  • 使用help查看指令并且用HELP+关键字查看详细操作;

    USE

  • USE 数据库名 来切换到指定数据库;

    SHOW

  • SHOW databases 查看所有数据库; SHOW tables 查看当前数据库中所有的表的列表;

    CREATE

  • CREATE database 数据库名 创建数据库

    SELECT

  • SELECT 选中的列 FROM 表名;
    其中被选中的列可以有多个; 可用通配符 “#” 选中表中所有的列;
    • DISTINCT 关键字用来返回指定列中不同的值;
    • LIMIT 关键字用来限制返回的行数; LIMIT num1,num2 返回第num1行,共num2行
      PS: 第一行标号为 行0; 若行数不够则返回现有行数
    • 完全限定的列名/表名 表名.列名 OR 数据库名.表名

      ORDER BY

  • ORDER BY 列名
    • 可对多个列进行排序,但排序时优先级按列的先后顺序;
    • ORDER　BY 一般按升序排列,使用 DESC 关键字用来降序排列;

    PS: DESC只对直接位于关键词前的列有效;

    WHERE

  • 根据WHERE子句中的 过滤条件 进行过滤;
  • WHERE子句在FROM子句后给出;
  • WHERE子句操作符
    操作符 意义
    = 等于
    <> OR != 不等于
    > OR < 大于/小于
    >= OR <= 大于/小于等于
    BETWEEN AND 在两数之间
    IS NULL 检查是否为空值
  • 用单引号来限制字符串;
    AND / OR / IN / NOT
  • AND 操作符
  • OR 操作符 优先 AND,然后是OR;
  • （） 用来指定优先级 ;
  • IN 操作符 用来指定括号范围（num1,num2,$\cdots$） 内数据;
  • NOT 操作符 用来排除 指定数据;

    LIKE

  • 指示WHERE利用通配符匹配
    通配符
  • % 表示任意位数的任意字符;
  • _ 表示一位任意字符;
  • PS:
    • 不要过度使用通配符;
    • 不要将通配符放在开始处;
    • 仔细注意通配符位置以避免错误搜索结果

HTML学习

HTML常用标签

结构标签

  1. <h1>-<h6> 标题标签
  2. <p></p> 段落标签 必须用段落标签才能分段
  3. <br/> 强制换行标签 是单标签

    文本格式化标签

  4. <strong></strong>or<b></b> 加粗标签
  5. <em></em>or<i></i> 倾斜标签
  6. <del></del>or<s></s> 删除线标签
  7. <ins></ins>or<u></u> 下划线标签

    无语义（盒子）标签

  8. <div></div> division 分割标签，独占一行
  9. <span></span> span 跨度标签，一行上可有多个

    图像标签

  10. <image src=”URL” alt=”替换内容” title=”鼠标在图片上显示的文本” width=”宽” height=”高”/>

    图像标签有多个属性，不分先后顺序，但必须在图像后面。

    相对路径：同一级路径直接文件名；下一级补完后续路径；上一级用”../“

    绝对路径：较少使用

    超链接标签

    外部链接

  11. <a href=”网址” target=”_blank”>文字</a>

    target值=”_self”当前页面打开or”_blank”新页面打开

    内部链接

  12. <a href=”相对路径” target=”_self”>文字</a>

    下载链接

  13. <a href=”文件地址”>文字</a> 文件格式要为”.zip”or”.exe”

    网页元素链接

  14. <a href=”网址” target=”_self”>网页元素</a>

    锚点链接

  15. <a href=”#名字”>文字</a> <h3 id=”名字”></h3> 链接将跳转到id与href相同的地方

    注释标签

16.<!–注释内容—>

  • 特殊字符
    • 特殊字符 代码
      空格 &npsp
      < &lt
      > &gt
      & &amp
      * &times
      / &divide
      平方 &sup2
      立方 &sup3
